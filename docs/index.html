About GitHubPro

GitHubPro is a project born out of my passion for exploring and leveraging GitHub's potential. Inspired by the desire to create a tool that enhances the GitHub experience, it provides users with valuable statistics and insights to empower their development journey. The development of GitHubPro is a milestone in my Holberton School journey, serving as a Portfolio Project to showcase my skills and commitment to the developer community.

This project is part of my coursework at Holberton School. You can find the GitHub repository for the project here.
